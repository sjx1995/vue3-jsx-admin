# vue3-jsx-admin

基于vue3和vite的管理界面模板

## Feature

- vue@3.0
- vite@2.0
- vue-router@4.0
- vuex@4.0
- jsx
- tailwindcss@2.1
- ant-design-vue@2.0

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
